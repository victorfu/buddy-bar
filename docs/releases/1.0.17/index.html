BuddyBar Release Notes

BuddyBar 1.0.17

April 11, 2026 · View on GitHub

Added

  • Added richer local usage history that matches Claude's official stats view, including daily activity, token totals, active days, and streak context.
  • Added session model and permission-mode details so active Claude sessions are easier to understand at a glance.
  • Added an ASCII art preview mode in the Buddy Gallery.

Improved

  • Improved Claude login recovery from API auth errors with an in-app claude auth login flow that opens cleanly from usage recovery.
  • Improved popover activation and foreground behavior when BuddyBar opens session, recovery, and login surfaces.
  • Improved session detail readability with clearer last-prompt formatting.

Fixed

  • Fixed the API auth recovery transition so the login popover reopens at the correct width instead of showing clipped or partially rendered UI.
  • Fixed Claude login command handling to use the current claude auth login arguments consistently.
  • Fixed debug app signing so local builds support system features more reliably during development.
